# Club public page — default hero asset generation prompts

Use these prompts when generating the stock hero images referenced in `src/lib/club-hero-default-assets.ts`. Export files to `public/assets/club-hero-defaults/` using the filenames listed in `public/assets/club-hero-defaults/PLACEHOLDER_ASSETS_TODO.md`.

All assets are intended to work with **`HeroImageTint`** (grayscale base + club primary overlay). Prefer neutral, desaturated bases.

---

## Asset 1: Neutral football team huddle hero

**File id:** `football-team-huddle-neutral`  
**Suggested filename:** `football-team-huddle-neutral.png`

**Prompt:**

Create a premium, realistic but slightly stylized football club hero image showing youth or amateur football players standing in a close team huddle from behind. No visible faces, no identifiable people, no logos, no text, no brand names. The players should wear plain white or very light neutral jerseys so the image can later receive a dynamic club-color overlay. Background should be a football pitch with soft stadium or training ground atmosphere. Composition should leave empty space on the left side for website hero text. Lighting should be cinematic but natural, with high contrast and clean edges. Style should be suitable for a modern SaaS club website. Export with transparent or very dark neutral background if possible. 16:9 ratio.

## Asset 2: Neutral football pitch hero

**File id:** `football-training-pitch-neutral`  
**Suggested filename:** `football-training-pitch-neutral.png`

**Prompt:**

Create a clean wide-angle football training pitch hero image with no people, no logos and no text. The scene should feel like a local sports club training ground in Europe. Keep the base colors mostly neutral, white, grey and desaturated green, so a dynamic club-color overlay can be applied later. Leave negative space on the left for text. Premium SaaS website hero composition, soft light, modern, clean, realistic. 16:9 ratio.

## Asset 3: Youth football action hero

**File id:** `youth-football-action-neutral`  
**Suggested filename:** `youth-football-action-neutral.png`

**Prompt:**

Create a dynamic but non-identifiable youth football action scene, viewed from distance or from behind, with players in plain white neutral kits. No faces, no logos, no text. The image should be suitable as a public club website hero. Keep the subject mostly white and neutral so a selected club color overlay can tint the image. Add realistic pitch background, cinematic sports lighting, clean composition, and empty space on the left for headline text. 16:9 ratio.
